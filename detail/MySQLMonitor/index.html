    MySQLMonitor¶

    MySQL实时监控工具（黑盒测试辅助工具）

    使用：¶

    java -jar MySQLMonitor.jar -h 127.0.0.1 -user CoolCat -pass mysqlmonitor

    环境：¶

    在以下环境中运行通过

    OS JAVA MySQL status
    MacOS 1.8 8.x ok

    环境下运行通过 理论上均可运行 如有问题或修改意见 请点击===>问题反馈

    功能¶

    • 自动开启MySQL日志 实时显示程序执行过的语句

    • 自动安装依赖

    应用场景：¶

    https://xz.aliyun.com/t/3875
